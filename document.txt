ALERT TRACK - INTERVIEW CODING CHALLENGE
========================================

SETUP
-----
- Run: npm install
- Run: npm run dev
- Open: http://localhost:3000
- Edit: src/app/page.tsx

REQUIRED
--------
- Click Alert 1 → alert appears in TOP-LEFT corner
- Click Alert 2 → alert appears in TOP-RIGHT corner
- Click Alert 3 → alert appears in BOTTOM-LEFT corner
- Click Alert 4 → alert appears in BOTTOM-RIGHT corner
- New alerts appear at the BOTTOM of the stack
- Each alert has a progress bar showing remaining time
- Alert auto-dismisses when progress bar reaches 0
- Hovering over an alert PAUSES the timer
- Moving mouse away RESUMES the timer

NICE TO HAVE
------------
- Manual dismiss button (X) on each alert
- Smooth animations on appear/disappear
- Different colors per alert type
- Timestamp on each alert
- Alert counter per corner

TIME: 45-60 minutes
